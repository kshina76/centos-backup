## 色々な問題の解き方
### パターンを考える問題(論理的思考)
- 方法
  - 「全ての組み合わせ」または「特定の組み合わせ」を考える
  - 組み合わせから何らかの規則性を導き出す
- 例
  - 容疑者A、B、Cがそれぞれ自分以外の人を犯人だと言った。Bだけは嘘をついていることがわかった。犯人はBか？
    - 全ての組み合わせを考える
      - A -> B、B -> C、C -> A
      - A -> B、B -> A or C、C -> B
      - AまたはCに集中するパターン
    - 辻褄が合うのはBが犯人の場合なので、Bが犯人

### ウミガメのスープ(水平思考、論理的思考)
- 方法
  - 前提を概念化した質問で確認
  - 概念化した質問で明らかになったものに対して、4W1Hで横方向に広げて、パターンを発想していく
  - 問題文に対して4W1Hで横方向に広げて、パターンを発想していく
  - Whyで縦方向に広げて、深堀りしていく
  - 定型質問を使ったパターン分析

## ウミガメのスープのコツ
- 例題
  - 小さな女の子が両親に「決して地下室の扉を開けてはいけない、開けたら見てはいけないものを目にする」と注意されていた。しかしある日、両親が出かけている間に女の子は地下室の扉を開けてしまった。果たして、女の子が見てはいけなかったものとは何だろうか？
- 解答
  - 女の子はその日、言いつけを破って生まれてはじめて地下室の扉を開き、外の世界を目にした。両親の手によって地下室の中に閉じ込められていたのは、その女の子だったのだ。
### 1. 問題文から単語を抜き出して前提条件をとりあえず絞る(名詞、動詞、名詞+動詞の文)
- 小さな女の子
  - 女の子である必要はある？
  - 人間？
  - 小さなというのは子供ということ？、背が小さいということ？、体が小さいということ？
    - 子供である必要はある？
  - 性別は重要？
  - 国籍は日本と考えていい？
- 両親
  - 両親は関係ある？
  - 両親は悪いことをしている？
- 地下室の扉
  - 地下室である必要はある？
- 開けたら見てはいけないものを目にする
  - 誰かが監禁されている？
  - 両親が物を隠している？
  - 両親が生き物を隠している？
  - 開けることで女の子に危害が加わる？
  - 犯罪が関係している？
  - 見られてしまうと誰かに危害が加わる？
- ある日
  - ある日というのは関係ある？、いつでも成り立つ？
  - 春？、夏？、秋？、冬？
- 両親が出かけた
  - 両親が出かけた場所は重要？
- 女の子が地下室の扉を開けた
  - 開けなければいけない理由があった？
  - 単純な好奇心から開けた？
### 2. 「本当に？」を問う
- クリティカルシンキングを行う
- 例
  - 「本当にパソコンを使わなかったの？」
  - 「本当に食べ物を食べなかったの？」
### 3. 5W1Hに沿って条件を増やしてみる
- 　「1」ではブレインストーミングのように発想を重視したが、ここでは5W1Hの形式に沿って行うことを重視する
### 4. 前提条件をさらに深掘りしていく
- 両親が何かを隠している？
  - 物を隠している？
  - 生き物を隠している？
    - 哺乳類？、爬虫類？、両生類？、霊長類？
      - 人間？
        - その人間は血が繋がっている？
        - 子供？大人？
  - 概念的な何かを隠している？
### 5. 変数の値を変えてみる(オズボーンのチェックリストとか使えるかも)
- 「靴」という変数があるなら「スニーカー」や「ヒール」といった定数を代入して考えてみる
- 「死因」という変数があるなら「事故死」や「他殺」や「自殺」といった定数を代入して考えてみる
### 6. 一つの変数の値を変えると他の変数が変わるものに注目
- 「スニーカーとヒール」という変数を変えると「身長」という変数が変わる
  - 次に「身長」が変わることで何が起きるかを考える

## メモ
- 上位の概念を質問して、掘り下げていくのが良い
  - 洋服？、靴？(×) -> 身につけるもの？(○)
  - 飲み物に毒が入っている？(×) -> 飲み物に何かが入っている？
  - 運動はテニスですか？(×) -> ラケットを使う運動ですか？、球技ですか？
- 中々思い浮かばなかったら、大前提の質問をするのもいいかも
  - そもそも人間なのか？
  - 現実世界の話なのか？
  - etc
- 「勘違いをしている？」という質問は効果的
  - 「オクラ入り」 <-> 「お蔵入り」などが問題になっていたりするから
- why？を使って垂直に深堀りする
  - whyでどんどん深堀りしながら、水平思考でアイデアを広げていくという一般的な問題解決と同じように行うといいかも
  - 泣いた(なぜ？) -> 悲しくて泣いた、嬉しくて泣いた、面白くて泣いた
- 5W1Hを使って名詞、動詞に対して具体化していく
  - 仕事 -> 芸能関係の仕事？、サラリーマン？、商売？
  - 甘いもの -> お菓子？、飲み物？、グミ？
  - 死んだ -> 他殺？、自殺？、事故死？
  - 掃除 -> 拭き掃除？、掃き掃除？、ゴミ拾い？
  - 飲む -> 水を飲む？、誤嚥？、薬を飲む？
  - 泣いた -> 悲しくて泣いた、面白くて泣いた、嬉しくて泣いた
- 定型質問
  - 職業の設定は関係ある？
    - 公務員？
    - サラリーマン？
    - スポーツやってる？
    - 経営者？
    - フリーター？
    - テレビに出ることがある？
  - 登場人物は人間？
  - 年齢は関係ある？
  - 性別は関係ある？
    - 戸籍上男？
    - 戸籍上女？
  - 国籍は関係ある？
  - 現実世界の話？
  - 仮想世界の話？
    - アニメの世界？
    - ゲームの世界？
    - 映画の世界？
    - 童話の世界？
    - SFの世界？
  - 場所
    - 屋内？
    - 屋外？
    - 地上？
    - 空？
    - 宇宙？
    - 水上？
  - 犯罪は関係ある？

## 様々な思考法をビジネスに応用する
1. 前提とゴールを決める
  - Why型思考、フェルミ推定(前提の決め方)を用いる
  - ゴールはなるべく概念化したもので
    - 概念化の例: 馬 -> 移動するもの
2. 前提を用いて、ゴールを目指すためには、どのような方法があるかを考える
  - 水平思考を用いる
    - 4W1Hなど
  - 例: スケート靴を履いて移動、何かを転がしてその上に乗って移動、ブースターを人間にくくりつける
3. アイデアの妥当性をチェックする
  - 論理的思考(Why型)を用いる
  - 調査、分析などをして論理的に筋が通っているかを確認する
