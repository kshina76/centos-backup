# タグ一覧
## id, class
- 要素に名前をつけることができる
- idは一つの要素に対して、classは複数の要素に同じクラス名を付けることで使える
- idにアクセスする方法は「#要素名」
- classにアクセスする方法は「.クラス名」

```html
<タグ id="id名"> </タグ>

<タグ class="同じclass名"></タグ>
<タグ class="同じclass名"></タグ>
```

## table, caption, tr, th, td
- 表を作ることができる
- colspan, rowspan属性を使うと、表を連結することもできる
