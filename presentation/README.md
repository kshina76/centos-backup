# 気を付けること  
・スライドのサイズ(比率)は16:9で  
・スライドマスター->フォント->フォントのカスタマイズ->英数字をSegoe UI、日本語をメイリオする。  
・本文はメイリオ18pt以上を使う。見出しは20ptとか本文より少し大きくしてもいいかも。  
・パワポのコンポーネントはthmxファイルをパワポにテーマをインポートして拾ってくる。  
・テンプレは、template.pptxから。
