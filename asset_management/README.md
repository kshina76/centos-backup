### 額面給与、課税給与、手取り
- 額面給与
  - 何も引かれていない状態の給与
- 課税給与
  - 「額面給与 - 控除額」
  - この状態はまだ税金は引かれていないことに注意
  - マッチング拠出は控除額に含まれるので、後述する所得税と住民税を節税できる
- 手取り
  - 「課税給与 - (所得税+住民税)」
- https://ten-navi.com/hacks/salary-7-7642
- https://www.rokinren.com/kigyonenkin-support/company_dc/matching.html

### 確定拠出年金
- 企業型(以下のどちらかのパターン)

  1. 企業が出してくれる掛け金 + 自分で出す掛け金(任意)
  2. 企業が出してくれる掛け金

- 個人型(iDeCo)
  - 従来は企業型の確定拠出金に加入していると入れなかったが、最近は入れるらしいからよく調べる
    - 月額20000円とか23000円とかかな？

- 確定拠出年金の特徴
  - 運用益に対して税金がかからない
    - 得た利益をそのまま運用に回せるから複利投資の効力をフルで受けれる
  - 企業型で自分から掛け金を出せるなら出したほうがいい
    - 給与を受け取らないで直接運用に回すから、所得税を節税できる
    - https://www.rokinren.com/kigyonenkin-support/company_dc/matching.html
  - 大まかに「元本保証型」と「投資信託」に二種類から選ぶ
    - 多分「投資信託」にフルベットして、投資信託の中でも「株式:債権=50:50」にして、その中でも「米国資産:欧州資産:日本資産=50:30:20」というように分散させるのがいいと思う

## NISAと積み立てNISA
- 一般NISAか積み立てNISAどちらかを年単位で選択する
### 基本
- NISAは利益に税金はかからない
- 本来かかるはずの約20%が非課税になるのがNISAの最大のメリット

![2021-02-16 1 22のイメージ](https://user-images.githubusercontent.com/53253817/107971228-90168e00-6ff5-11eb-8225-c796e4a9e9ef.jpg)

### 一般NISAと積み立てNISAの違い
- 非課税枠とは、NISA口座でその年の1~12月にて投資できる額の合計
- 一括と積み立てというのは、非課税枠の120万を一括で払ってしまうかどうかということ
  - 積み立てなら12*10万
- 非課税運用期間は、購入した時点から何年非課税になるかというもの
- 奥さんの分も合わせてできるので年に80万円分を非課税にできる
- 子供の分のJrNISAを使えるかも

### ふるさと納税
- ふるさと納税とは
  - どこの市町村でも寄付できる
    - 自分が住んでいるところである必要はない

  - 手順は後述する
  - 例: 50000円の寄付の場合、2000円を引いた48000円が翌年の住民税に当てられて、返礼品を受け取れる
- ワンストップ特例制度
  - サラリーマンで、確定申告が不要の制度
- ふるさと納税のおすすめ先（日用品、食料品）

  - 福岡県北九州

  - 兵庫県加東市

  - 愛媛県西条氏
    - 16000円の寄付額で、乳児用オムツ(4000円相当)
- 楽天ふるさと納税
  - 楽天が運営する、ふるさと納税サイト
  - SPUやお買い物マラソンのポイントは、翌月末までに使わないといけない
  - ポイントの使い道
    - 楽天PAYで、「コンビニ、飲食店、楽天でんき、など」または、楽天モバイル、など
  - SPU（7%）+ お買い物マラソン（9%）+ 5と0のつく日(2%) = 計18%(17倍)のポイントがつく

      - 楽天会員: 1

      - 楽天市場アプリ経由で買い物: 0.5
      - 楽天ブックス: 0.5
      - 楽天モバイル: 1
    - お買い物マラソン
      - お買い物マラソン開催中に、ショップ数に応じたポイントがつくサービス（10個のショップで購入したら10%つく）
      - 1000円以上の寄付額で、異なる10個の自治体で寄付すると、1ショップとしてカウント（自治体も1ショップとしてカウントしてくれる）
      - ワンストップ特例制度の場合は、5個までしかカウントできないので、その他の5ショップは日用品を購入して埋めれば良い（本、シャンプー、トイレットペーパー、など）
    - 5と0の付く日
      - 2%のポイントがつく
    - 例: ふるさと納税で10000円寄付したら、18%なので1800円分のポイントがつく

- 定期預金（まとまったお金用）は、ボーナスから捻出する
  - まとまったお金は、医療の立て替えなどのため

## 医療費
### 社会保証(3割負担)
- 社会保証額を払っているので、医療費の自己負担は3割に抑えられる
- 3割でも高額になる場合は、高額療養費制度が使える
- 所属している健保によっては、高額療養費制度の限度額よりも抑えることができる
  - 自分が所属している健保は、毎月20000円以上は医療費がかからない
### 高額療養費制度
- 医療費の負担が毎月20000円以上はかからない
  - 高額療養費制度だけだと、毎月80000円（所得による）ほどが自己負担になる
  - うちの会社の健保は、医療費の負担が毎月20000円以上かかった分を、給付金として支給してくれる
### 自社健保
- 1レセプトで、自己負担が20000円を超えたぶんは、給付金として支給
  - レセプト1件とは、1人、1ヵ月、1医療機関単位
    - 通院の場合でも、ひと月の合算が20000円を超えたら支給されるということ
- 同一世帯において同じ月に21,000円を超える自己負担が2件以上生じた場合にも、支払われるっぽい
- 入院時
  - ベッド差額は無料
    - 病院側の都合の個室などは、個人がお金を払う必要はないとされている
  - 食事代は毎日かかる
  - その他雑費もかかる

## 保険
- 基本的には、リスクがかなり高いものに対しては入るようにする
  - 火災、事故、etc
  - 医療に関しては、社会保障が充実しているため、入らなくても良い
### 生命保険
- 結婚して、子供ができたら生命保険を多めにかければ良い
  - それ以外のときには、入る必要はない
### 医療保険
- 高額療養費制度と自社の健保で賄うので、必要ない
- 先進医療を受けたいなどのことがあれば、入る必要はあるが
  - オブシーボなどは、認可されているため社会保障で受けられる
### 自動車保険
- 入る必要あり
### 火災保険
- 入る必要あり

### [手順](https://www.youtube.com/watch?v=OXcAYKREc2I)
1. 源泉徴収票を用意する
  - 「2」で入力する内容が書いてある
2. シミュレーションをする
  - [控除額計算機](https://www.furusato-tax.jp/about/simulation?header_guide)
  - [早見表](https://www.furusato-tax.jp/about/easy_simulation?header_guide)
  - 注意点
    - 新卒はどうする？
      - 給与所得を「8ヶ月分+ボーナス」をして計算をする
    - iDecoをやっている場合はどうする？
      - 小規模企業共済掛金の金額に入力
      - 積み立てNISAは給与控除がないので気にしなくて良い
3. シミュレーションの額を寄付
  - 寄付する時期
    - 楽天のポイントが最大化できるとき
    - 3ヶ月に1回のペースなどで寄付しないと、冷蔵庫がパンパンになったりする
  - 楽天ふるさと納税を使用する
  - ワンストップ特例制度
    - 確定申告不要
    - 申し込む自治体は5箇所まで。同じ自治体なら1箇所でカウント。違う自治体に10箇所などを申し込みたい場合は、確定申告が必要になる
    - 申し込んだ自治体ごとに申請書を提出
    - 同じ自治体であっても、その都度送付する必要あり
    - マイナンバーなども送付するので注意。寄付した年の翌年1月10日までに、書類を寄付先の自治体に送る
    - 住所が変更になったら、全ての自治体に変更届を提出
  - 注意点
    - 送付先: 誰でも良い
    - 注文者情報: 住民票の住所となっているか確認する。間違えると控除できなくなる可能性
    - ワンストップ特例申請書: 申請書をもらう
4. 支払う住民税から「寄付額-2000円」が引かれて納税されているか確認
  - 2020年1月〜12月の年収などで算出した場合、2021年の6月〜5月に支払う住民税から「寄付額-2000円」が引かれる
  - 現住所の自治体から住民税の納税書が届く
