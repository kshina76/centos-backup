## ふるさと納税
- 額面550万で、69000円の控除なので、低く見積もった67000円をふるさと納税に使う
- 後述した、ふるさと納税の返礼品でちょうど67000円
- ワンストップ特例制度を使用するので、5つの自治体まで（同一の自治体なら複数でもカウントされない）
### [手順](https://www.youtube.com/watch?v=OXcAYKREc2I)
1. 源泉徴収票を用意する
  - 「2」で入力する内容が書いてある
2. シミュレーションをする
  - [控除額計算機](https://www.furusato-tax.jp/about/simulation?header_guide)
  - [早見表](https://www.furusato-tax.jp/about/easy_simulation?header_guide)
  - 注意点
    - 新卒はどうする？
      - 給与所得を「8か月分+ボーナス」をして計算をする
    - iDecoをやっている場合はどうする？
      - 小規模企業共済掛金の金額に入力
      - 積み立てNISAは給与控除がないので気にしなくて良い
3. シミュレーションの額を寄付
  - 寄付する時期
    - 楽天のポイントが最大化できるとき
    - 3か月に1回のペースなどで寄付しないと、冷蔵庫がパンパンになったりする
  - 楽天ふるさと納税を使用する
  - ワンストップ特例制度
    - 確定申告不要
    - 申し込む自治体は5箇所まで。同じ自治体なら1箇所でカウント。違う自治体に10箇所などを申し込みたい場合は、確定申告が必要になる
    - 申し込んだ自治体ごとに申請書を提出
    - 同じ自治体であっても、その都度送付する必要あり
    - マイナンバーなども送付するので注意。寄付した年の翌年1月10日までに、書類を寄付先の自治体に送る
    - 住所が変更になったら、すべての自治体に変更届を提出
  - 注意点
    - 送付先: 誰でも良い
    - 注文者情報: 住民票の住所となっているか確認する。間違えると控除できなくなる可能性
    - ワンストップ特例申請書: 申請書をもらう
4. 支払う住民税から「寄付額-2000円」が引かれて納税されているか確認
  - 2020年1月〜12月の年収などで算出した場合、2021年の6月〜5月に支払う住民税から「寄付額-2000円」が引かれる
  - 現住所の自治体から住民税の納税書が届く
- 参考文献
  - [手順など](https://www.youtube.com/watch?v=OXcAYKREc2I)

## 投資
- 長期投資なら100%株式でも良い
- 人生が後半になって、資産を守りたい場合は債券多め
- 株式（米国インデックスファンド）: 国内債券（インデックスファンド）+ 円= 7 : 3
- 円は、生活防衛資金分を残しておけば良いかも
### 積み立てNISA
- 楽天証券を活用
  - 楽天カード決済で、設定をしておくと毎月1月に、勝手にやってくれる
- まとまったお金が必要になったら、必要な分だけを売る
- [出口戦略](https://www.youtube.com/watch?v=Mxtw4vsS-AI)
- [積み立てNISAの始め方](https://www.youtube.com/watch?v=clsCi6J3iR0)

  - 全世界を100か、米国株式を100で良いかも。それでもDCと合わせて60%を米国に投資することとなるので、割合は悪くない気がする
  - 積み立てNISAでリバランスするのが面倒なので、DC側だけでリバランスするようにするため。

### 無痛分娩
- 病院によっては、費用がかからないところもある

- 無痛分娩ってこんなに安いのになぜ普及していない？？
### 出産一時金
- 出産する医療機関が、産科医療保障制度に加入している場合、1人当たり42万円もらえる
### 出産時の付加給付金
- 出産時に自社の健保の場合、20万円追加でもらえる
### 出産時（無痛分娩でも可）の医療費控除
- 所得税と住民税から医療控除額に税率をかけた分が、節税になる
- 医療費控除額（上限200万円）＝［1年間の医療費の合計額］－［保険金などの補てん金額］－ 10万円
- 補てん金額は、出産一時金と付加給付金も入ると思う
### 出産手当金（勤務先へ健保から）
- 出産予定日42日～出産後56日まで（いわゆる産休の期間）、1日につき日給の3分の2が支給
### 育児休業給付金（雇用保険から）
- 産休が明けると、育児休業に入りますが、その育児休業中も給付金が支給
- 原則、子どもが満1歳を迎えるまで休業前の賃金の67％相当額が支給
  - 育休中の社会保険料（健康保険料、厚生年金）は免除だから、実際の手取り額はそれほど変わらないからすごい！！
### パパ・ママ育休プラス
- 父親も育休を取得すると、給付金の支給期間が2か月延びる制度
### 児童手当
- 所得が960万以下の家族は、10000〜15000円が、「赤ちゃん〜中学生」まで貰える
### 子どもの医療費助成（市区町村）
- 乳幼児の医療費の自己負担分を援助する制度
- 東京23区はすべての区で中学3年生まで医療費の保険診療分が全額無料で、所得制限もない
### 幼児教育無償化制度（市町村）
- 共働きを対象にした制度で3歳～5歳の保育園・こども園・幼稚園の保育料が無償
- フルタイムだけでなくパートタイムでの就業も対象
