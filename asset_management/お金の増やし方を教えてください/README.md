# お金の増やし方を教えてください 書籍メモ

## スキルは一番の投資
- 企業が倒産しようが、国が財政破綻しようが自分のスキルはずっと役に立つ

## 個人向け国債「変動10年」
- 安定を重視するなら国債
  - 国債は国にお金を貸してあげて、それに対して利息をつけて返してくれるという仕組み
  - 国債は戦争で負けたり、財政破綻したりしない限りは安全

## 投資信託
- 生活費を除いたお金は定期預金しないで、投資信託に回してお金に働いてもらう
### 投資信託とは
- プロが株を選んで運用してもらうもの
- 株の詰め合わせセットみたいな感じ
- 資産を比較的安全に保管できるものとも言える
  - 資産は投資をする会社に保管されているわけではなくて信託銀行に保管されているから
### 投資信託の買い方
- 販売手数料(買うときに一度だけかかる)と運用管理手数料(持っている間、払い続ける)が少ないものを選ぶ
  - 販売手数料はネット証券の口座で購入することによって手数料を下げられて、場合によってはゼロ
  - 運用管理手数料は手数料が低い投資信託を選択する
- 投資成績はあてにしない
- おすすめ
  - 上場インデックスファンドTOPIX
  - ニッセイ外国株式インデックスファンド
### 投資信託はなぜ儲かるか
- ドルコスト平均法という考え方
### インデックスファンドとアクティブファンド
- アクティブファンド
  - プロ（人）が株や債券を選んで買うやつ
  - 手数料が高いのでナシ
- インデックスファンド
  - 日経平均とかダウ工業平均とかニュースでやってる指標に連動する投資信託
  - 手数料が安いのでアリ
- 実績の違い
  - 過去の実績を見るとプロが運用するアクティブファンドの平均がインデックスファンドに勝ったことはほとんどない
### TOPIXと日経平均
- TOPIXは日経平均より会社の数も多いし、会社の規模も考慮されているから、よりバランスのとれた指数
### いつ売るのか？
- お金が必要になったとき
  - 病気になったり、老後にお金が必要になったときに、必要な分だけ売ればいい。それまでは持っておき、ひたすら運用する
- お金が必要になったときは、もし自分が買った金融商品の値段が購入金額より下がっていても、躊躇なく売る

## やるべき投資
- NISAを使ってインデックスファンドに投資することで5年間税金がかからないので絶対に利用するべき
### いくら投資するかを決めるステップ
1. 資産から当面の生活費を引く
2. 「1」で余ったお金を「安全資産」と「リスク運用資産」に分割する
3. 「安全資産」で個人向け国債を買う
4. 「リスク運用資産」でインデックスファンドを国内と海外を半々で買う

### 投資信託
- 上場インデックスファンドTOPIX（国内株式の投資信託）
  - 特徴
    - 運用管理手数料（ランニングコスト）が安い
    - 購入手数料（イニシャルコスト）が無料
    - 海外の企業へ分散投資ができる
- 〈購入・換金手数料なし〉ニッセイ外国株式インデックスファンド（外国株式の投資信託）
  - 特徴
    - 「ETF」といって、上場している株式と同じように購入できるので、ネット証券を利用すると、売買の手数料が安い。１００万円分を買っても数百円程度
    - 運用管理手数料（ランニングコスト）がかなり安い（年率０・１％程度）
    - 運用資金量と取引量が多く安定している
### 国債
- 個人向け国債「変動10年」
  - 特徴
    - 長期金利に連動するので、銀行の金利に比べて大きく損することが少ない
    - 個人向け国債は最初の１年はおろせないが、それ以降は過去２回分（１年分）の金利を払えば換金できる
    - 銀行に比べて安全な保管場所と言える

## 生命保険とガン保険
- 入った方がいい場合
  - ネットの保険会社で、子供が自立するくらいまでの10年～20年、最低限の期間に掛け捨て型で、『残される家族の人数』×『１０００万円』くらいの死亡保障だけの特約のないシンプルな保険に入るといい
    - 遺族年金があるから1000万円くらいで十分
  - 掛け捨てなら安いし、保険会社が潰れたら別の保険会社に変えればいいだけの話だし
  - 先進医療を受けたいなら入ったほうがいいかも
- 入らなくていい場合
  - 家族がいないなら入らなくていい
  - 高額療養費制度があるから
