- 自社とライバルは、ともにシャア50%、月の売上高は100万円で拮抗している。自社・ライバルともに製品価格は100円、品質差はない。自社は値下げをするべきかどうか？
  - Level3で、相手に「その行動をしたらまずいよ」というシグナルを送ることが重要。そしてもしその行動を取られても大丈夫なように備えておく
